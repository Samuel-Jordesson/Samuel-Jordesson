## Sobre Mim

Desenvolvedor apaixonado por transformar ideias em produtos reais, funcionais e bonitos. Busco sempre a união entre arquitetura de qualidade e interface moderna, criando experiências digitais memoráveis.

- Stack: **Next.js + Node.js + Prisma + Tailwind CSS**
- Especialidade: SaaS, CMSs visuais, painéis admin, dashboards
- Atualmente: Desenvolvendo sistemas de agendamento, portais de conteúdo e gestores visuais
- Foco: Entregar MVPs com design limpo, performance e funcionalidade de verdade
